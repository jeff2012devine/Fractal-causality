    <h2>Abstract</h2>
    <p>
      <em>Fractal Causality™</em> proposes that the universe is not a one-off event but a self-similar, cyclical process in which local collapses seed new expansions. Black holes serve as transformation chambers where holographically stored information undergoes a loop-quantum-gravity–style bounce and conformal flip, re-expressing compressed two-dimensional data as new three-dimensional initial conditions. 
      Version 3.3 presents a minimal mathematical model, stability criteria, and falsifiable predictions across the CMB, large-scale structure, gravitational waves, and 21 cm cosmology.
    </p>

    <h2>How to Test Fractal Causality</h2>
    <ol>
      <li><b>CMB Analysis</b> — Compare measured <i>n<sub>s</sub></i>, <i>f<sub>NL</sub></i>, and tensor ratio <i>r</i> with model predictions.</li>
      <li><b>Large-Scale Structure</b> — Search for log-periodic oscillations in P(k) and BAO shifts with DESI/Euclid.</li>
      <li><b>Gravitational Waves</b> — Detect stochastic GW background with LISA at mHz, slope n<sub>t</sub> ≈ –0.04.</li>
      <li><b>21 cm Cosmology</b> — SKA can test predicted oscillatory features ΔP/P ~ 10⁻³ at z ~ 10–20.</li>
      <li><b>Black Hole Mass Function</b> — Compare clustering at 10¹³–10¹⁵ M<sub>☉</sub> to predicted imprints.</li>
    </ol>
