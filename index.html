Plain-Language Summary

The hypothesis: the universe carries a hidden rhythm — a repeating log-periodic pattern — that shows up across scales. From galaxy clustering to gravitational-wave event timing, atomic-clock drift, and even brain signals, the claim is that a single underlying “oscillator” may leave consistent fingerprints. The goal here is simple: look for those fingerprints. If they’re not there, the hypothesis fails.

Purpose: This page presents Fractal Causality as a testable hypothesis and a reproducible scaffold. It is not a final theory. Below is a falsifiable prediction.
Headline Prediction: If Fractal Causality is true, then independent datasets — galaxy distributions, gravitational-wave event timing, long-baseline atomic-clock drift, and EEG signals — will exhibit a common log-periodic frequency band within a pre-specified tolerance. If no such band appears across ≥2 independent domains under the same analysis, the hypothesis is falsified.

How to Test the Hypothesis

Quick Test (Data-curious / Non-specialist)

  1. Pick a time-stamped dataset (e.g., public gravitational-wave event times, long atomic-clock drift series, or open EEG).
  2. Transform time to log(t) (choose a sensible epoch; sensitivity analysis below).
  3. Fit X(t) = A + B·cos(ω·log(t) + φ) and scan for ω with Lomb–Scargle or FFT on log-time.
  4. Generate surrogate series (phase-randomized / shuffled intervals) and repeat.
  5. If the real data’s peak power at some ω* consistently exceeds surrogates (p<0.01 after multiple-test correction), you’ve found a candidate log-periodic frequency. If not, that dataset is a fail.

Technical Protocol (Specialists)

  1. Targets (choose ≥1; mixing across domains encouraged):
    • GW timing: LIGO/Virgo event catalogs.
    • Cosmology: galaxy clustering / BAO residuals.
    • Metrology: atomic clock drift / frequency stability time series.
    • Neuro: open EEG datasets for wide-band log-periodic structure.
  2. Signal model:
    Fit on log-time or log-scale: x(t) = A + Σ B_k · cos(ω · log(t) + φ_k).
  3. Nulls & surrogates:
    • Interval shuffles / phase randomization.
    • 1/fα colored-noise nulls.
    • Systematic nulls (calendar, detector uptime, etc.).
  4. Statistics:
    • Peak-finding in log-frequency; permutation p-values.
    • Bayes factor: log-periodic vs. null.
    • Cross-domain consistency test.
  5. Decision rule (falsifiable):
    • Falsify if: no domain shows significant peaks beyond nulls or discovered peaks are inconsistent.
    • Survives if: ≥2 domains show consistent peaks replicable by external teams.

Note: This is a hypothesis. Criteria are strict so it can be proven wrong.
